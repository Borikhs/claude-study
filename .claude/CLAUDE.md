# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 스터디 목적

이 저장소는 Claude 및 Claude Code를 깊이 이해하고 실무에 활용하기 위한 스터디 공간입니다.

주요 목적:
- **Claude Code 공부**: Claude Code의 기능, 워크플로우, 활용법을 직접 실습하며 학습
- **최신 정보 팔로업**: Anthropic의 업데이트, 새로운 모델·기능·API 변경 사항을 지속적으로 추적

스터디 운영 방식:
- 학습 내용은 **노션**에 정리하여 문서화
- **정기 발표**를 통해 팀원과 학습 내용 공유 및 토론

## About This Repository

개인 또는 팀 스터디 목적의 실험·학습 공간으로, 고정된 기술 스택은 없습니다. 언어나 프레임워크에 관계없이 다양한 실험 프로젝트가 추가될 수 있습니다.
